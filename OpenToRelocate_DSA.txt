================= MASTER PATTERN GUIDE =================

Sliding Window →
Use when:
- substring / subarray
- longest / smallest window
- K constraint

Two Pointer →
Use when:
- sorted array
- pair sum / palindrome

Prefix Sum →
Use when:
- range sum
- subarray sum = K

Binary Search →
Use when:
- sorted array
- monotonic condition
- answer space search

Heap →
Use when:
- kth smallest/largest
- top K frequent

Backtracking →
Use when:
- permutations / combinations
- recursion tree

Graph (BFS/DFS) →
Use when:
- grid / matrix
- connected components

DP →
Use when:
- overlapping subproblems
- optimize recursion
