================ SYSTEM DESIGN =================

[HLD - High Level Design]

1. Social Media Feed Design
- Requirements: post creation, feed generation, likes/comments
- Approach:
  - Fanout-on-write (push to followers)
  - Fanout-on-read (generate at request time)
- DB: NoSQL (posts), cache (Redis)
- Scaling: partition by userId
- Optimization: pagination, caching

--------------------------------------------------

2. Notification System
- Async system using Kafka
- Retry mechanism + DLQ
- Idempotency to avoid duplicate notifications
- Channels: email, push, SMS

--------------------------------------------------

3. URL Shortener
- Generate unique short key (Base62 / hashing)
- Store mapping (short → long)
- DB: key-value store
- Handle collisions
- Add analytics (click count)

4. Rate Limiter
